## SchoolJoon

특정 학교의 구성원들이 해결한 백준 문제들을 분석하는 프로젝트입니다.

백준 문제들로 다양한 재밌는 기능을 추가할 예정입니다.

일단은 건국대 대상으로 처리할 예정

크롤링할 때 서버에 무리가 가지 않도록 2초 딜레이 설정

(프로젝트명은 임시입니다.)
